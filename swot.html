Strengths

Сильные стороны

01

Вертикальная интеграция

Единственный продукт, объединяющий бизнес-план + каталог оборудования + штатное расписание + расчёт ремонта + прайс услуг в одном дашборде. LivePlan генерирует шаблоны, Dentira продаёт расходники — никто не собирает полный конструктор.

02

Edge AI на Jetson

Qwen 2.5:32B на Jetson AGX Orin (275 TOPS, 64GB). Нулевые затраты на inference, работа без интернета, контроль данных. По прогнозу IDC, к 2027 году 80% AI inference будет локальным.

03

Реальные цены рынка

Конкуренты — абстрактные "industry benchmarks". SVITA — реальные цены оборудования, подрядчиков, зарплат с конкретного рынка (Варшава). Разница может составлять сотни тысяч PLN.

04

5 ролей в экосистеме

Админ, владелец, подрядчик, поставщик, сотрудник — каждый видит свой интерфейс. Поставщик заполняет каталог, подрядчик конкурирует ценой, владелец видит итог.

05

Авторасчёт ROI

Кресла × пациенты × дни × чек = выручка. Минус расходы = маржа, payback. Изменение любого параметра — мгновенный пересчёт всего бюджета.

06

Franchise-ready

Конструктор масштабируется на любой тип бизнеса. FranConnect — $500-2000/мес за управление существующими сетями. SVITA запускает с нуля.

07

$0/мес стоимость

GitHub Pages (free) + Supabase free tier + Jetson (~$2000 однократно). ProAI за 2 года: $2,400-4,776. LivePlan Premium: $720/год.

08

Экосистема SCYRA.AI

7 AI-агентов (ADM, SCH, COM, VIS, DOC, VOI, PAT) на том же Jetson. От конструктора до операционной системы клиники.

Weaknesses

Слабые стороны

01

Прототип v3

Single HTML + inline JS, данные в localStorage. Нет реальной авторизации, нет 3D модели. Конкуренты — зрелые продукты с годами разработки и тысячами пользователей.

02

Узкая ниша

Стоматология, Польша. TAM: $208M. ProAI обслуживает все отрасли глобально. Масштабирование требует новых баз данных для каждого региона и вертикали.

03

Bus factor = 1

Вся разработка — один человек (CEO). Конкуренты имеют команды из десятков разработчиков, маркетологов и поддержки.

04

Нет мобильного приложения

60%+ бизнес-пользователей работают с мобильных. У Bizplanr, Upmetrics, IdeaBuddy — мобильные приложения или mobile-first.

05

Нет внешних интеграций

LivePlan синхронизируется с QuickBooks/Xero. ProAI генерирует pitch deck. SVITA — изолированная система без API и экспорта.

06

Ручные каталоги

Dentira/ALARA — реалтайм каталоги 10,000+ позиций с автообновлением цен от 13+ поставщиков. SVITA требует ручного наполнения.

07

Нет track record

0 пользователей, 0 отзывов, 0 success stories. LivePlan — "30 лет экспертизы". ProAI — "3600+ кейсов".

80% inference локально к 2027 (IDC). Когда облачные конкуренты столкнутся с ростом затрат на API — SVITA уже автономна.

06

B2B с поставщиками

Threats

Угрозы

01

AI-прорыв конкурентов

LivePlan, ProAI активно внедряют AI-агентов. С командами и инвестициями — за 6-12 мес добавят вертикальные решения для медицины.

02

Облачные гиганты

Microsoft Copilot, Google Gemini могут интегрировать бизнес-планирование в свои экосистемы. Мгновенно изменят рынок.

03

EU AI Act

Медицинские AI — высокий уровень регулирования. Сертификация, аудит, документирование увеличат стоимость и время выхода на рынок.

04

Pearl AI / Overjet

Зрелые стоматологические AI-платформы с FDA-одобрением и тысячами клиентов. Если добавят startup planning — преимущество аудитории.

05

Устаревание edge-железа

Jetson AGX Orin мощный сегодня, но через 2-3 года появятся быстрее. Облако обновляется автоматически.

06

Низкий барьер входа

"AI бизнес-план для стоматологии" = 2-3 месяца на OpenAI API + шаблоны. Преимущество — в глубине, не в идее.

07

Зависимость от Qwen

Экосистема открытых LLM быстро меняется. Смена модели = перенастройка промптов. Облако просто переключает API.

08

Макроэкономика

39% предпринимателей оптимистичны в 2026. Спад → меньше открытий клиник. Инфляция обесценивает базы цен.

💎 Главное преимущество

Ни один продукт на рынке не объединяет AI-ассистента + реальные каталоги оборудования + расчёт ремонта + штатное расписание + финмодель + мультиролевой доступ в одном конструкторе. Конкуренты либо генерируют абстрактные документы, либо решают одну узкую задачу. SVITA — единственный путь "от идеи до операционной модели" с реальными данными.

⚠️ Главный риск

Прототип против зрелых продуктов с тысячами пользователей. Стратегия "первый в нише" работает только при быстром занятии позиции — до того, как крупные игроки добавят вертикальные решения для медицины.
