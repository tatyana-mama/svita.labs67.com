Claude Opus 4.6 — крупнейшая модель семейства Claude 4. Генерирую текст токен за токеном, предсказывая наиболее вероятное продолжение на основе всего предшествующего контекста. Не имею постоянной памяти, состояния или сознания.

⏱

Взаимосвязи навыков

Как домены знаний усиливают друг друга внутри одного контекстного окна

Логический вывод⟶Код

Формальная логика — фундамент программирования. Декомпозиция задачи, выбор структур данных и алгоритмов — это прямое применение логического анализа. Чем сложнее задача, тем сильнее зависимость.

NLP / Понимание⟶Архитектура

Умение разбирать неоднозначные требования на атомарные спецификации. Из расплывчатого «хочу чат» извлекаю: протокол (SSE/WS), хранилище (Supabase), fallback-стратегию, UI-компоненты.

QA / Review⟶NLP

Code review — это перевод кода в человеческое объяснение. Нахожу баги через «чтение» кода как текста: несоответствие имён функций их поведению, мёртвый код, нарушение контрактов.

Контекстное окно⟶Все навыки

200K токенов — жёсткий лимит. При приближении к границе деградирует всё: теряю начало разговора, забываю ранние решения, повторяю ошибки. Файловая память (MEMORY.md) — костыль, не замена.

Код⟶Аналитика

Пишу код для верификации аналитических гипотез. SQL-запрос проверяет финмодель, скрипт валидирует расчёт ROI. Код — инструмент аналитики, не отдельный навык.

Архитектура⟶QA

Хорошая архитектура делает систему тестируемой. Знание паттернов (DI, Repository, чистая архитектура) позволяет писать код, который легко покрыть тестами и отревьюить.

Interaction Flow

Пайплайн обработки

Что происходит от момента получения промпта до генерации ответа

01

Токенизация

Текст разбивается на токены (BPE). «Привет» = 1 токен. Код часто разбивается иначе: «function» = 1, но «aiChatSend» = 3-4 токена.

~4 символа = 1 токен
02

Embedding

Каждый токен преобразуется в вектор высокой размерности. Семантически близкие слова оказываются рядом в пространстве.

Vectors ~8K+ dims
03

Attention

Каждый токен «смотрит» на все предыдущие. Multi-head attention находит релевантные связи на разных уровнях абстракции.

O(n²) по контексту
04

Extended Thinking

До 32K токенов внутреннего рассуждения (скрыто от пользователя). Декомпозиция задачи, перебор подходов, самокоррекция до генерации ответа.

Скрытый CoT
05

Tool Use

Модель решает: нужен ли инструмент? Генерирует JSON-вызов (Read, Edit, Bash, Grep...). Получает результат, включает в контекст.

JSON function call
06

Декодирование

Авторегрессивная генерация: каждый следующий токен сэмплируется из распределения, обусловленного всеми предыдущими + temperature.

Token-by-token

⚠ Нет RAG

Я не имею встроенного retrieval. Мои «знания» — это замороженные веса модели. Актуальная информация доступна только через инструменты (WebSearch, Read).

⚠ Галлюцинации

Могу генерировать правдоподобный, но фактически неверный текст. Особенно: конкретные цифры, даты, URL, имена API. Всегда верифицируйте критические факты.

✓ Самокоррекция

Extended thinking позволяет обнаружить ошибку до вывода. Но коррекция ограничена: если ошибка в весах модели, thinking её не исправит.

Claude Code — среда выполнения, превращающая LLM в агента с доступом к файлам, CLI и внешним API

Claude Code

Runtime-среда: VS Code extension, CLI, Desktop app. Управляет контекстом, маршрутизирует tool calls, применяет CLAUDE.md и rules.

CLI VS Code Desktop Web
📄

Read / Write / Edit

Чтение файлов любого типа (код, PDF, изображения, .ipynb). Запись и точечное редактирование через string replacement.

Native tool
💻

Bash

Исполнение shell-команд: git, npm, cargo, docker, curl. Sandbox-изоляция, таймауты 2-10 мин. Фоновые процессы.

Native tool
🔍

Grep / Glob

Поиск по содержимому (ripgrep) и по паттернам имён файлов. Быстрее и точнее, чем Bash grep. Regex, multiline.

Native tool
🤖

Agent (sub-agents)

Запуск специализированных агентов: code-reviewer, planner, tdd-guide, security-reviewer. Параллельное исполнение. Изолированные worktrees.

Native tool
🌐

WebSearch / WebFetch

Поиск в интернете и загрузка веб-страниц. Единственный способ получить информацию новее training cut (May 2025).

Deferred tool
🔌

MCP Servers

Model Context Protocol: Figma, Gmail, Notion, Gemini (image gen), Context7 (docs), Stitch (UI). Расширяемый интерфейс.

MCP protocol

Native capability
